This project is a MSVC wrapper of openslam_gmapping
(https://github.com/ros-perception/openslam_gmapping)

main.cpp - a port of ROS Node slam_gmapping/slam_gmapping 
(https://github.com/ros-perception/slam_gmapping)

The rest of files are pulled from openslam_gmapping
The only dependency is OpenCV_2.4.9, for visualization.

Make sure the opencv .dll files are included in system path.

This project is compiled and tested under Microsoft Visual Studio Community 2013.
